API DOCUMENTATION

  • GET / = root api-info page
  • GET /userdata/get = get all userdata from database
  • POST /userdata/register = post userdata to database to register
  • POST /userdata/login = check if email and password match, returns id and login status (true/false)
  • PUT /userdata/change/name/:id = change name of user with specific id
  • DELETE /userdata/delete/:id = deletes user with corresponding id
